Bem-vindo à Minha API REST
BOIAPI

Esta é a página inicial da documentação da boiAPI. Use os endpoints abaixo para interagir com nossos serviços.

Ao acessar a raiz vc tera esta pagina

GET /
Host: api.boiapi.com.br/
User-Agent: curl/7.64.1
HTTP/1.1 200 OK
Content-Type: application/json
{ "message":"bem vindo a minha API" , "dica":"de uma olhada nos endpoints" }

Ao usar o endpoint /arroba vc tem um GET geral com
todos os valores

Use para ter uma visualizacao ampla dos dados, conhecer
a estrutura dos dados ,e entender como estao organizados

GET /arroba/
Host: api.boiapi.com.br/arroba/
User-Agent: curl/7.64.1

Ao usar os endpoints /arroba e /estado combinados vc tera os
valores referentes a aquele estado

Use para visualizar os dados de um estado especifico
Os possiveis valores sao :

SP MG GO TO

RS RJ MS MT

PR SC ES BA AC

MA AL RO RR PA
GET /arroba/estado
Host: api.boiapi.com.br/arroba/MG
User-Agent: curl/7.64.1 { "date":"2025-12-22", "animal":"garrote nelorado", "arroba":"331.5", "estado":" MG", "regiao":" MG" }, { "date":"2025-12-23", "animal":"novilha", "arroba":"317.84", "estado":" MG", "regiao":" MG" }

Ao usar os endpoints /arroba e /animal combinados vc tera os
valores referentes a aquele animal em diferentes estados

Use para ter uma visualizacao geral dos valores de um animal especifico

os possiveis valores sao :

  • boi
  • vaca
  • garrote
  • novilha
  • bezerro
  • bezerra
GET /arroba/animal
Host: api.boiapi.com.br/arroba/boi
User-Agent: curl/7.64.1 { "date":"2025-12-22", "animal":"boi", "arroba":"325,00", "estado":"RJ", "regiao":"" }, { "date":"2025-12-22", "animal":"boi", "arroba":"311,50", "estado":"BA", "regiao":"" }

Para realizar um POST vc deve pode usar um cliente de API
de sua preferencia, ou usar um comando com curl semelhante a este :

POST
>> curl http://127.0.0.1:8000/update --json '{ >> "date":"2025-12-23", >> "animal":"bezerro desmamado", >> "arroba":"446.15", >> "estado":"SP", >> "regiao":"SP" >> }'
