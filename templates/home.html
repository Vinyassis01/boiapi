PecuáriaData API

Cotações atualizadas de Boi Gordo e Reposição em todo o Brasil.

Obter Chave de API Documentação Técnica

1. Autenticação

Acesse as rotas de modificação e filtros avançados utilizando um token Bearer.

POST /v1/auth/token
curl -X POST "https://api.pecuaria.com" \ -H "Content-Type: application/json" \ -d '{ "client_id": "seu_id", "client_secret": "sua_chave_secreta" }'

2. Exemplo de Consulta

Recupere preços do Boi Gordo filtrando por estado (MT, MS, SP, etc).

GET /v1/cotacoes
curl -X GET "https://api.pecuaria.com" \ -H "Authorization: Bearer SEU_TOKEN_AQUI"

3. Referência Rápida

Método Endpoint Descrição Acesso
GET /v1/estados Lista estados disponíveis. Público
GET /v1/cotacoes Consulta valores com filtros. Privado
POST /v1/cotacoes Atualizar dados de mercado. Admin
